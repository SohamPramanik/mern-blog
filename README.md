📝 MERN Blogging Platform

A full-stack blogging web application built with the MERN stack (MongoDB, Express, React, Node.js).
Users can create posts, upload media, like and comment on blogs, manage profiles, and switch between light/dark themes.

🚀 Features

Authentication: Register, login (JWT), logout confirmation

Blog Management: Create, edit, delete, and view posts

Media Upload: Upload images/videos with posts

Social Features: Like posts and add comments

User Profile: View and manage personal posts

UI: Responsive design with light/dark theme

🛠 Tech Stack

Frontend: React, React Router, Axios, CSS
Backend: Node.js, Express.js
Database: MongoDB, Mongoose
Auth: JWT
File Upload: Multer
